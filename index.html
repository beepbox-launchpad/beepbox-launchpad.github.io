Dogebox2:

Dogebox2 is a mod of JummBox by DogeisCut. It adds custom fm operator frequencies, the note range and invert wave effects, and was the first mod to add note flashing. It was released on September 9, 2023.
You can find Dogebox2's Wiki Page here. Which contains extensive information about the mod.

Nepbox:

Nepbox is a mod of ModBox created by Neptendo. It intends to modernize ModBox back into standing with the other mods, by adding quality of life features. It was released on January 11, 2024.
You can find Nepbox's Wiki Page here. Which contains extensive information about the mod.

Unbox:

Unbox is a mod of UltraBox by un_conventional. Unbox recreates the Duty Cycle from Sandbox and makes 0-100% pulse width range possible. It was released on January 20, 2024.
You can find Unbox's Wiki Page here. Which contains extensive information about the mod.

shitbox4:

shitbox4 is a joke mod of beepbox by choptop84. Shitbox4 begrudgingly aims to make BeepBox as unbearably bad as possible. It was released on March 13, 2024.

BariBox™:

BariBox™ is a mod of UltraBox by Bariani. BariBox adds a "corruption" feature, which essentially randomizes the placement of notes, patterns, and the instrument types used. It was released on March 14, 2024.
You can find BariBox's Wiki Page here. Which contains extensive information about the mod.

LemmBox is a mod of UltraBox that was created by HappyLemonLime and was released on January 7th of 2025. Its main appeal is its customization of the max values found in the editor, and extra chip waves.
